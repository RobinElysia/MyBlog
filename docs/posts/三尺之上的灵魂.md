# 三尺之上的灵魂
</font>
<font style="text-align:center">

qwp--2024.12.8
</font>
<font size=5em>

悬在三尺之上灵魂啊

你可曾见到过生命在蔷薇中绚丽绽放？

就当作是我来时的赠礼，当作是一把烈火燃烧后的残存。

我看到你，三尺之上的灵魂

可是我将笨重的砸向地面，

那是白云下的“雨中舞者”，

是飘零落下的枫叶，将最后的微风衬托起。

我三尺之上的灵魂啊

我见到你如见到彼生彼世

在你的回响中我听到我从未听到过的悦耳之声

灰烬将复燃、落叶将再次乘风而起

生命在次次轮回中，变得熟练

伴随你的传来的节奏，我将赞美：

> 昨日，我换取痛苦带来的欢乐
> >
> 今日，我沉浸充实带来的满足
> >
> 明日，我聆听未知带来的福音
> >
> ——三尺之上的灵魂--2024.12.8

我的灵魂啊，你是我终将为之期待尊重的——

在两个无限中，你将存在于之中充当主导,

在生命的圆舞中划出一道道音符,

这将是我生命中最期待的时刻，

我三尺之上的灵魂
</font>
