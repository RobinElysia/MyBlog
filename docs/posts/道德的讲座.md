# 道德的讲座
</font>
<font style="text-align:center">

“上帝已经死了，那些妄图以虚假崇高名义继续蒙蔽我们双眼的，是牧羊人，是旧时代的掘墓者，是背后世界论者……所以我的鹰与蛇将同我下降，直至我与“我”的知己一同回归。”
</font>
<font style="text-align:right">

——qwp 2024.8.3
</font>

&emsp;曾有位智者，他能将所有的道德作为理所应当，把所行之事作为必须的负担，教听众行事犹如退化后的骆驼，俯下身子，甘愿任人宰割； 也曾有位愚者，他传授听众判决既定的命运，面对理所应当的义务坚定说“不”，高下立判成为过去，而今已然成为雄狮……

&emsp;但孩童，他怎知何为是非对错，你若问他：“遇难者不救，是否有违道德？”他不知…

&emsp;你若问他：“践踏他人者，是否有违道德？”他还不知…

&emsp;你若问他：“背信弃义者，是否有违道德？”他更不知…

> 孩子是纯洁的，是遗忘，是一个新的开始，一个游戏，一个自转的车轮，一个肇事的运动，一个神圣的肯定。
> >
>                                     ——查拉图斯特拉如是说--2024.8.3

&emsp;但孩童啊，他的心大底是清而明的，是权衡利弊的。当他真正胜过自己内心不偏不倚时，智者和愚者都将消失。是的，到那时我也将沉静的走上山去，去往内心的那片净土，独享久违的孤独。

&emsp;最后的最后，智者成为了愚者，而愚者成为了我。 在回去的路上，恍惚间，人群中有人大声地叫到：“他说的是对的！上帝真的死了！我们今后只能依靠我们的大地！” 我不以为然，这是他们应该知道的，而不是后知后觉。 我的同情也从未消散，因为我怜悯那些奴隶之人，他们将自己的大理约束在小理之上，被可憎的牧羊人夺取，被牧犬和创造祂的人一一欺骗。 我继续沉默地向山上走去，让他们其中的“我”代我向你们传以超人之思。

> 你就这样结结巴巴地说：“这是我的善，我爱它，它是完全如此使我满意，我就是要单独这种善。”
> >
> “我要它，并不想当它是一位神的律法，我要它，并不想当它是人间的规章、人间的必需品：对于我，我不要它成为指向超越大地之世界和天上乐园的路标。”
> >
> “我爱它，它是世上的道德：在它里面没有多少聪明，更没有多少万人共通的理性。”
> >
> “可是，这只鸟儿啊！到我这里来做巢：因此我爱它，抱它，————现在，它在我这里孵它的金蛋”
> >
> 你应当这样，像这样结结巴巴地赞美你的道德。
